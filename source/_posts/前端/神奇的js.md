短路表达式、三元表达式

```js
a = b && 1
// 相当于
a = !!b ? 1 : b
if (b) { a = 1 } else { a = b }

a = b || 1
// 相当于
a = !!b ? b : 1
if (b) { a = b } else { a = 1 }
```

取整(不四舍五入)

```js
~~2.33 === 2
2.33 | 0 === 2
2.33 >> 0 === 2
Math.floor(2.33)
```

```js
'0' == 0 // true
0 == [] // true

'0' == [] // false
```

```js
{} + [] === 0 // true
```

```js
parseInt(0.0000005) === 5
// 如果 parseInt 第一个参数不是字符串，会将其转换成字符串，之后在解析
// 小于 10-6 的浮点数以指数表示
// parseint 从 float 的指数法中提取整数
String(0.0000005) // => '5e-7'
parseInt(5e-7) // => 5
parseInt('5e-7') // => 5

parseInt(1111111111111111111111) // => 1
parseInt(999999999999999999999) // => 1
```
