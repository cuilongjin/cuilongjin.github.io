快速删除node_modules 文件夹

第一种方法：（最方便，删除速度超快）

``` 
  cnpm install rimraf -g 
  
```

第二种方法：

```
  rmdir /s/q your_app_dir
```

/s 是代表删除所有子目录跟其中的档案。 
  /q 是不要它在删除档案或目录时，不再问我 Yes or No 的动作。 

第三种方法：

```
  rm -f /node_modules
```
