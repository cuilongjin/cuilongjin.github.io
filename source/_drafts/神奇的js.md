短路表达式、三元表达式

```js
a = b && 1
// 相当于
a = !!b ? 1 : b
if (b) { a = 1 } else { a = b }

a = b || 1
// 相当于
a = !!b ? b : 1
if (b) { a = b } else { a = 1 }
```
